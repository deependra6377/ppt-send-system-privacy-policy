Privacy Policy

PPT Send System respects user privacy. This application is designed to allow users to send emails using their own Gmail accounts.

Gmail Data Usage

We request access only to the Gmail sending permission (gmail.send) to send emails on behalf of the user.

  • We do NOT read Gmail inbox messages.
  • We do NOT store Gmail message content.
  • We do NOT share Gmail data with third parties.

Data Storage

We securely store OAuth refresh tokens only to maintain Gmail sending functionality. Users can revoke access at any time from their Google Account settings.

User Control

Users have full control over their data and may disconnect Gmail access at any time.
