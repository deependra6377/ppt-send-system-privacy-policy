Privacy Policy

PPT Send System respects the privacy of its users. This application is designed to help sales representatives send presentation files (PPT/PDF) to clients using their own Gmail accounts.

Gmail Data Usage

Our application uses Google OAuth 2.0 to request access to the gmail.send scope only.

  • We do NOT read Gmail inbox messages.
  • We do NOT store Gmail message content.
  • We do NOT modify or delete emails.
  • Emails are sent only when the user explicitly initiates the action.

Data Storage

We store OAuth refresh tokens securely to maintain Gmail sending functionality. These tokens are used only to send emails on behalf of the authenticated user.

User Control

Users can revoke Gmail access at any time from their Google Account security settings. Once access is revoked, the application will no longer be able to send emails.

Third-Party Sharing

We do not share Gmail data, email content, or user information with any third party.
